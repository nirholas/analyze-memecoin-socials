# analyze-memecoin-socials

Research repository for analyzing memecoin social sentiment and price action.

## Structure

- **`chart/`** — Interactive chart visualizations (pump-fun-sdk chart, social sentiment, tweet×price)
- **`data/`** — OHLCV price data (1-min, hourly, daily candles) for pump-sdk token
- **`scripts/`** — Data fetching scripts (GeckoTerminal OHLCV, X cashtag scraper, profile scraper)
- **`tweet-price-charts/`** — Original upstream project ([rohunvora/tweet-price-charts](https://github.com/rohunvora/tweet-price-charts)) — Tweet-Price correlation analyzer with Next.js web app
